# O3DE-Gems
This repo exists to help you more easily find and add Heathen's O3DE Gems to your project. To add the repo open the O3DE Project Manager select Gems
<img width="1516" height="770" alt="image" src="https://github.com/user-attachments/assets/dd2faff9-ce9b-4bee-9215-3e09648dda0f" />
From the menu on the upper right side select "Show Gem Repos" and click "Add Repository"
<img width="1520" height="769" alt="image" src="https://github.com/user-attachments/assets/b89003dd-4cbd-42ce-8882-039695cafd7d" />
Use this URL `https://github.com/heathen-engineering/O3DE-Gems/main/src`
Once complete you can return to your Gems and search "Heathen" to see a list of all of Heathen Engineering's free Gems
